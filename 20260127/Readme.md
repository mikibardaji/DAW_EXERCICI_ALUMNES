# Feina per casa no de codificació 

Per casa, mira els primers minuts d'aquest concurs, perque el dijous codificarem [aquest joc](https://www.telecinco.es/television/a-la-carta/20250728/alla-tu-programa-completo-video_18_016231909.html). Sol cal mirar, fins que ha obert 3 o 4 caixes i truca la banca al telefon per fer una oferta.

Pujar un fitxer txt amb els metodes que  creus que s'haurien de codificar, per poder simular el concurs.


un exemple podria ser el següent
**Exemple: Aplicació de música**

Funcionalitats que s’haurien de programar:
Guardar una canço:

→ Preguntar una canço i guardar-la a la posició elegida

Mostrar totes les cançons:

→ Mostrar tot el array amb les cançons guardades

Seleccionar una cançó concreta:

→ Preguntarem al usuari una posició

Reproduir canço:

→ Et poasara el nom de la primera canço.

I si algú s'anima, perque el concurs no necessita cultura general, us podeu apuntar [al casting](https://www.gestmusic.es/castings/casting-alla-tu/)
